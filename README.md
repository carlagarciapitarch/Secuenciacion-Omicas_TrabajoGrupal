# Secuenciacion-Omicas_TrabajoGrupal
En este repositorio GitHub se encuentran los archivos utilizados para la resolución de la actividad grupal de asignatura Secuenciación y Ómicas de próxima generación. El objetivo era analizar la expresión diferencial de genes de 3 personajes con obesidad (obesidad 2) y 3 personajes con normopeso.

Los pasos que hemos seguido han sido
1. Control de calidad (directorio Control_de_Calidad).
2. Pseudoalineamiento (directorio Salmon).
3. Estudio de la expresión diferencial de genes (directorios DESeq2 y EdgeR).
4. Estudios de enriquecimiento (directorio enriquecimiento).

Los archivos design.csv y transcrito_a_Gen eran dados en el enunciado para la resolución de la actividad.
